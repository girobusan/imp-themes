IMP! Themes

This site is dedicated to IMP! themes, you can view themes in development in sampler.

How to install

Choose the theme in Sampler, then right-click the get css link, then you have two options:

  • Save the css near your IMP!, then enter css name in IMP! interface under view CSS location, or...
  • Replace your style.css with downloaded file (if you haven't changed location of view CSS, and do not use CDN version)

Themes are based on:

  • Gutemberg CSS
  • SimpleCSS
  • TypeBase

...and other open source projects. You can use all of them without restrictions.
